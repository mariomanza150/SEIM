This guide will help end-users (students, coordinators, managers, admins) navigate and use the SEIM system.

## Getting Started
1. Go to http://localhost:8000
2. Register as a student or log in with your credentials.
3. Use the dashboard to access your applications, documents, and notifications.

## Student Workflow
- Create a new exchange application using the multi-step form.
- Upload required documents (PDF, images, etc.).
- Track your application status and respond to feedback.
- Receive notifications for status changes.

## Coordinator/Manager Workflow
- Review submitted applications.
- Approve, reject, or request changes.
- Add comments and track application progress.
- Verify uploaded documents for integrity and validity.

## Document Upload & Verification
- Only allowed file types and sizes are accepted.
- All files are scanned for viruses and checked for integrity.
- Download and preview documents as needed.

## Notifications
- Email and in-app notifications are sent for key events (submission, approval, rejection).
- Manage your notification preferences in your profile settings.

## FAQ
- **Q:** I can't log in. What should I do?
  **A:** Use the password reset link or contact your coordinator.
- **Q:** My document upload fails.
  **A:** Check file type/size and your internet connection.
- **Q:** Who can see my application?
  **A:** Only authorized staff and yourself.
